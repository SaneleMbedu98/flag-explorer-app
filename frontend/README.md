# Flag Explorer Frontend

Frontend for the Flag Explorer App, built with React and Tailwind CSS.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Initialize Tailwind CSS:
   ```bash
   npx tailwindcss init
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Testing
Run tests with:
```bash
npm test
```
